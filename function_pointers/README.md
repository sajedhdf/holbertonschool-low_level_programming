# 0x0E. Function pointers

## 0. What's my name

**Mandatory**

Write a function that prints a name.

### Files

- `0-print_name.c` — contains the function `print_name` that prints a name using a function pointer.
- `function_pointers.h` — header file containing the prototype of `print_name`.

### Usage

A test file `0-main.c` can be used to test your function (not included in the repository for Holberton checker).
