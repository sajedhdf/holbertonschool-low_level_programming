# argc_argv Project

This project contains several programs demonstrating how to use `argc` and `argv` in C.

## Files

### 0-whatsmyname.c
Prints the program name.

### 1-args.c
Prints the number of arguments passed.

### 2-args.c
Prints all arguments.

### 3-mul.c
Multiplies two numbers. Prints `Error` if arguments are missing.

### 4-add.c
Adds positive numbers. Prints `Error` if a non-digit appears.

## Compilation

Use this command for all files:

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 file.c -o executable
